## v1.8.0: window.close fallback修正 + Debug Console

### Issue
- Garoon（ns2.kyusanko.biz）でF1押下時に「Scripts may close only the windows that were opened by them」エラー
- スタックトレース: content.js:75 (fallback) → window.close()
- 原因: Service Worker休止 → chrome.runtime.id undefined → fallback発動 → window.close()はスクリプトが開いたウィンドウ以外閉じられない

### Fix
- window.close() fallback削除
- 代替: トースト通知「拡張が更新されました。ページを再読み込みしてください」
- reload/hardReload のみ location.reload() fallback維持（これは動作する）
- Service Workerリトライ: 1回50ms → 3回(0ms/100ms/500ms)バックオフ

### Debug Console (新規)
- debug.html: ポップアップ🐛ボタンから起動
- 機能: SW状態, Ping, アクションログ(50件), Key Test, バインディング表示, Debug Mode, Auto Refresh
- background.js: logAction()でアクション結果を記録、ping/getDebugInfo/clearDebugLogハンドラ追加
- content.js: debugMode時にF-key keydownイベントをconsole.log
